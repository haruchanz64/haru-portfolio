About Me

I'm a systems-oriented developer currently working as a backend developer at a social media startup. Having shipped multiple games with Unity and with growing experience in Godot, I gravitated toward the infrastructure layer — the tooling, architecture, and systems that make development faster and more reliable.

I specialize in backend API development with C# (.NET), while exploring Java (Spring Boot) and lower-level game development through Bevy and Raylib. I also build custom engine tooling and workflow automation for game projects.

I'm open to freelance work in backend APIs and custom engine tooling. If you have something to build, let's talk.
